# 🏦 Bank Account Management System – C++

This project is a functional Bank Account Management System developed in C++. It demonstrates fundamental programming concepts and robust data handling using file systems to simulate real-world banking operations.

## 🚀 Features

* User Management: Efficiently create and manage user profiles within the system.
* Client Database: Store and retrieve comprehensive client information using structured text files (Users.txt, Clients).
* Transaction Simulation: Handling basic banking operations to simulate a real-world environment.
* Data Persistence: Automatically saving all data to external files ensures that information is never lost after closing the program.

## 🧠 System Logic

* Record Conversion: Advanced functions to convert records into lines and vice versa using custom separators (#//#).
* File I/O: Reading from and writing to files to maintain a consistent database state.
* Data Structures: Implementation of struct for client data and vector for dynamic list management.

## 🛠️ Technologies Used

* Language: C++
* IDE: Visual Studio 2022
* Storage: File Handling (Text-based Database)
